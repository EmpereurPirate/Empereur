Empereur
A Python script to download models from Hugging Face model hub.

Requirements
Python 3.6 or higher
requests library
tqdm library
Installation
Clone this repository:

git clone https://github.com/EmpereurPirate/Empereur.git
Install the required libraries:

pip install requests tqdm
Usage
Run the download_model.py script with the name of the model you want to download:


python download_model.py ShinojiResearch/Senku-70B-Full
The script will download the model files to a directory named after the model.
